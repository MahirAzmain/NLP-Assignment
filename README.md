## NLP Assignment Question: IMDB Movie Sentiment Analysis

### Dataset
Use the IMDB movie review sentiment dataset containing 50,000 samples (25,000 positive and 25,000 negative reviews).

### Tasks

#### Classical Machine Learning Models
Train Logistic Regression (LR) and Support Vector Machine (SVM) classifiers.  
Use 300-dimensional Bag-of-Words (BOW) and 300-dimensional TF-IDF embeddings as input features.

#### Deep Learning Models
Train Bidirectional LSTM (Bi-LSTM) and Bidirectional GRU (Bi-GRU) models.  
Use 300-dimensional Word2Vec and 300-dimensional GloVe embeddings as input features.

### Data Split
Use an 80:20 ratio for training and testing.

### Evaluation Metrics
Report Precision, Recall, F1-score, and Accuracy for each model and embedding combination.
